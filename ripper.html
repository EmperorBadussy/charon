WHAT IS CHARON?

CHARON is the companion app to AETHER — your premium desktop music player. CHARON downloads music from Tidal and delivers it straight to your Navidrome library, where AETHER picks it up and plays it with stunning visualizers and binaural audio.

The Pipeline: Search Tidal → Download with CHARON → Navidrome scans → Play in AETHER

GETTING STARTED

  • Install Python 3.8+ and ensure it's in your PATH
  • Install tiddl: pip install tiddl
  • Install tidalapi: pip install tidalapi
  • Install ffmpeg and ensure it's in your PATH
  • Run tiddl once to authenticate with Tidal
  • Click the auth status indicator to check your connection

HOW DOWNLOADING WORKS

CHARON uses tiddl (a Python CLI tool) to download from Tidal. The download command is: tiddl url <URL> download -q master -p "<your music folder>". Tracks are saved to your configured download directory. After each download, CHARON can auto-trigger a Navidrome library scan so your new music appears immediately in AETHER.

QUALITY TIERS

  • MAX — MQA / HI_RES_LOSSLESS (up to 24-bit/192kHz)
  • LOSSLESS — FLAC 16-bit/44.1kHz (CD quality)
  • HIGH — AAC 320kbps
  • NORMAL — AAC 96kbps
